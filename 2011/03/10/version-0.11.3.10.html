Version 0.11.3.10

Release notes

  • Search widget is moved to toolbar. It also takes input focus at startup.
  • Search filter is smarter. Matched group shows all entries.
  • New Debian/Ubuntu package revelation-mime which allows to remove revelation and still have file associations, icons and registered mime-type. Just install revelation-mime and remove revelation.
  • Debian/Ubuntu packages now use lzma compression. Packages are 30% smaller now.
